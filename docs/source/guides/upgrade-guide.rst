=============
Upgrade Guide
=============


Introduction
============

This guide, intended for NLNZ Tools SIP Generation users, covers any issues related to upgrading versions.

The source for both code and documentation for NLNZ Tools SIP Generation can be found at
https://github.com/NLNZDigitalPreservation/nlnz-tools-sip-generation/.

Contents of this document
-------------------------

Following this introduction, the NLNZ Tools SIP Generation Upgrade Guide includes the following sections:

-   **Upgrade requirements** - Covers requirements for upgrading.

Upgrade requirements
====================

There are no upgrade requirements since the current version is the only version.
